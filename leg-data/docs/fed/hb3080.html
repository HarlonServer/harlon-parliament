Criminal Code Foundation Act

HB 3-080

Author: Westray

5 Sept 2023

Opening Declaration

  1. The purpose of this bill is to establish the criminal code of Harlon. This document will contain an exhaustive list of all crimes that can be prosecuted within the Federation of Harlon.

Information

  1. This bill will contain the criminal code, which outlines acts that are to be prosecuted on a criminal basis. Victims of these crimes may also file civil lawsuits for additional damages, such as a company harmed by fraud, if deemed applicable in future legislation.
  2. For efficacy of the system of justice, there will be the following enforcements:
  1. “Per Offence” summary enforcement shall involve punishments that can be made against any individual prior to trial by a prosecutorial or law enforcement authority. These punishments can be appealed to the court.
  2. “Maximum Sentence” indictable enforcement shall involve punishments that can only be made through a ruling by an officer of the court.
  1. The criminal code shall include the list of crimes, enforcement mechanisms, and details on how it may be amended in the future. The code is linked here under the title “1st Criminal Code of Harlon”

Implementation and timeline

  1. This act will be enforced one week after passage.
  2. Upon enforcement, the criminal code will be posted in full on the website in a place accessible to players for viewing.
